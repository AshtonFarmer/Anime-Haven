# Anime Haven

A phone-first installable anime tracker built for Ashton and Tanima.

## Features

- Separate libraries for two profiles
- Watched/in-progress and haven’t-started sections
- Season and episode tracking with finished-season history
- Alphabetical, recent, and franchise watch-order sorting
- Global search across both profiles plus AniList search
- Local offline saving, backup export, and backup import
- Installable Progressive Web App
- Luffy-inspired extending punch search, charge-up navigation, and a Dark Repulser-inspired scroll indicator

## GitHub Pages

The included workflow deploys the repository as a static GitHub Pages site. In repository **Settings → Pages**, choose **GitHub Actions** as the source if Pages is not already enabled.

## Data

Progress is stored locally in the browser. Use **Settings → Export backup** to transfer both profiles to another device.
